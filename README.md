## 🍱  项目简介

**sky-takeout-plus** 是一个面向外卖业务场景的综合管理系统，旨在模拟真实外卖平台的核心业务流程。系统采用主流 Java Web 技术栈，实现了**管理员端**与**用户端**的业务分离，具备良好的可扩展性与工程规范性。
